# Install Git
## To track changes and push your code to GitHub.
### Where we use: Local machine
### Command / Installation:
 ## Windows: Download and install from https://git-scm.com/downloads
 # Check installation:
# git --version

 # GIT  TO ADD FOLDERS /FILES
## Initialize Git using git init.
### Skip if already initialized.
### After that, to add files, git add. 
### After that, commit the changes, git commit m and the file name. Without commit, we can't push. 
### After that, we need to push to GitHub, git push origin main


# WHEN CRETED A NEW REPOSITORY

## When you are connecting to a new repository
### first we have to add a remote === git remote add origin https://github.com/yourusername/your-repo-name.git
### after git push -u origin main , 
## after that,
### git add .
### git commit -m "Updated test files"
### git push
